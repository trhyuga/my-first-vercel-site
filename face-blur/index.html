動画の中の顔を、最初から最後までぼかす。

指定した人だけ・指定した人以外・全員 の3モード。アップロードした動画はこの端末内だけで処理されます。

🎬
タップして動画または写真を選択
動画: MP4 / MOV / WebM・写真: JPG / PNG / WebP / HEIC など
動画は実時間の1〜2倍、写真は数秒で処理

プレビュー

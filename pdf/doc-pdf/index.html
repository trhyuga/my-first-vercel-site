プレビューをブラウザの 印刷ダイアログ → 「PDFとして保存」 で書き出します。
この経路がクライアント側で一番レイアウトを崩さない方法です。iPhoneは Safari の共有から「ファイルに保存 / プリント」→ 拡大(プリント画面でピンチアウト)→ 共有 → 「ファイルに保存」 で PDFにできます。

PPTX (PowerPoint) はクライアント側でレイアウトを正確に再現しきれないため、このツールでは対象外です。スライド資料は Google スライド / Keynote / PowerPoint の「PDFとして書き出し」を推奨。
